# Simulating-multi-soliton-solutions-of-the-nonlinear-Schr-dinger-equation-in-a-BEC
Study of nonlinear wave dynamics and soliton soliton collisions

## How to use the code
Copy and pase the code from the appendix into python and vary the parameters to explore collision dynamics

## What it does
The code explores how phase impacts collision dynamics between solitons, parameters such as collision angle, velocity and phase are chosen so that investigations into collisions can be explored.
